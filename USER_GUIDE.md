# Trading Journal App - User Guide

## 1. Sign In
- Open the app and sign in with Google or email/password.
- New users can create an account from the sign-in modal.

## 2. Add a Journal Entry
- Go to the `Journal` page.
- Choose the trade date.
- Drop or type the instrument.
- Enter PnL.
- Select profit or loss.
- Rate the trade.
- Add lessons, mistakes, and optional journal comments.
- Click `Save Entry`.

## 3. Use the AI Journal Assistant
- Open the floating AI assistant on the journal page.
- Describe your trade in plain language.
- Review the structured result.
- Confirm to save the entry.

## 4. Review Your Dashboard
- Open `Dashboard`.
- Use the date range filters to review a period.
- By default, the dashboard opens on the current month.
- View KPIs, chart trends, and recent trades.
- Ask for analysis only when needed.

## 5. Export Analysis
- Generate dashboard analysis.
- Use the PDF export option to print/save the report.

## 6. Trial and Renewal
- Trial users see access reminders during the trial.
- After a renewal, reminders show again only when 15 days or less remain.
- Use the renewal action when prompted.

## 7. Admin Usage
- Admin users can open the admin page.
- Review user access, approvals, and login activity.
- Approve payment requests from the admin panel.

## 8. Personalization
- Journal page layout is saved per user.
- Clock mode preference is saved per user.
- Daily tips are selected per user per day.

## 9. Language
- Use the language selector in the header to switch between English and Hindi.
