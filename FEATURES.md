# Trading Journal App - Feature List

## Authentication
- Google sign-in
- Email/password registration and sign-in
- Trial access tracking
- Login count and last-login tracking

## Journal
- Manual trade entry with:
  - date
  - instrument
  - PnL
  - profit/loss result
  - trade rating
  - lessons or mistakes
  - journal comment
- Custom in-app date picker
- Drag-and-drop layout for journal widgets
- User-specific layout persistence
- AI-assisted journal entry launcher
- AI chat saves the user message as journal context
- User-specific daily tip
- Digital / analog clock with saved user preference

## Dashboard
- Current-month default reporting
- Custom date range filters
- KPI cards for trades, PnL, win rate, and rating
- Cumulative PnL chart
- Recent trades list
- On-demand analysis panel
- PDF-friendly report export
- Trial / renewal visibility rules

## Subscription & Trial
- 30-day trial support
- Renewal request flow
- UPI payment support
- Renewal toast visibility based on trial / renewal status
- Renewal tracking with renewal count

## Admin
- User access overview
- Pending approval queue
- Renewal and trial status visibility
- Login count and last login visibility
- Admin notification banner for pending approvals

## Security
- Journal entry encryption at rest
- Auth-protected APIs

## Branding & UX
- Custom brand logo and favicon
- GitHub Pages SPA fallback for direct URLs
- Localized English and Hindi support
